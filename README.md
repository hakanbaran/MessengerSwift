# Talk To Me

- An application linked to your spotify account where you can listen to albums, playlists and tracks...
Powered by Spotify API...
Build with UIKit...

### Technologies used in the development of the application:
- UIKit
- AVFoundation
- MVVM
- TabBar
- TableView
- CollectionView
- CoreData
- SearchBar
- WebView
- SDWebImage
- Spotify API
